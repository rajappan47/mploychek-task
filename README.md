# MPloyChek Verification Dashboard

## 📌 Overview
This project is a Single Page Application (SPA) built using Angular and Node.js to simulate a digital background verification system.

## 🚀 Features
- Role-based Login (Admin / General User)
- Dashboard with user details
- Verification records table
- Status indicators:
  - Verified
  - In Progress
  - Discrepancy
- API delay simulation (async behavior)
- Admin user management (if implemented)

## 🛠️ Tech Stack
- Frontend: Angular
- Backend: Node.js 

## ▶️ Run Project

### Backend
cd backend  
node server.js  

### Frontend
cd frontend  
ng serve
